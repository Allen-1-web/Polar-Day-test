# Полярный день

Учебный проект: минимальный интернет-магазин одежды без сервера и без сторонних библиотек. Данные хранятся в `localStorage` браузера.

## Возможности (V1)

- Регистрация и вход по email и паролю
- Каталог товаров и добавление в корзину
- Оформление заказа с учебной формой «оплаты» (реальных списаний нет)
- История покупок для текущего пользователя

## Структура проекта

| Файл | Описание |
|------|----------|
| `index.html` | Разметка экранов |
| `style.css` | Стили |
| `app.js` | Логика и данные в браузере |

Каталог задаётся массивом `PRODUCTS` в `app.js` — его удобно дополнять и менять.

## Запуск

1. Откройте файл `index.html` в браузере (двойной щелчок или перетаскивание в окно браузера).

   Либо из папки проекта в терминале:

   ```bash
   npx --yes serve .
   ```

   Затем перейдите по адресу, который покажет команда (обычно `http://localhost:3000`).

2. Зарегистрируйтесь, добавьте товары в корзину, заполните форму оплаты и проверьте раздел «История покупок».

## Ограничения учебной версии

- Пароли и заказы хранятся только у вас в браузере, без шифрования — так в продакшене нельзя.
- Оплата симулирована; для настоящих платежей нужен банк или платёжный сервис и серверная часть.
- Данные не синхронизируются между устройствами и пропадут при очистке данных сайта в браузере.
