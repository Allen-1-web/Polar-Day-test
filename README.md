## Полярный День — интернет‑магазин одежды (V1, черновик)

Простейший одностраничный прототип интернет‑магазина одежды «Полярный День».

### Функциональность

- **Каталог товаров**: несколько демо‑позиций с ценой и доступными размерами.
- **Выбор размера**: у каждого товара есть селект размера; без выбранного размера добавить в корзину нельзя.
- **Корзина**:
  - добавление товаров из каталога;
  - изменение количества, удаление позиций;
  - отображение итоговой суммы.
- **Доставка / получение**: только **самовывоз** (выбор пункта самовывоза в форме).
- **Способ оплаты**: выбор между **наличными** и **безналичной оплатой**.
- **Оформление заказа**:
  - минимальный набор полей: имя, телефон, пункт самовывоза, способ оплаты;
  - необязательный комментарий к заказу;
  - валидация обязательных полей и наличие товаров в корзине.
- **Подтверждение заказа**: после успешной валидации выводится текст «Заказ подтверждён…», а в консоль браузера попадает объект с данными заказа.

### Технологии

- Чистый HTML + CSS + JavaScript (без фреймворков, без сборки).

### Запуск

1. Откройте файл `index.html` в браузере (двойной клик или «Открыть с помощью…»).
2. Для корректной работы в некоторых браузерах лучше поднять простой локальный сервер (например, `serve`, `http-server` или через встроенный сервер в редакторе).
